Comparing Dog Foods using Javascript

After a recent addition to the family, myself and my partner have spent some time researching dog foods. As she is in the scientific field she is very interested in the nuitritional value from each food but we could not find a site that compared foods very well.

It was interesting working on something to solve a particular problem and this is now something my partner uses when checking foods against each other. Potentially, this could be a useful future app!

This tables contents (other than the titles) are fully created and inserted within JavaScript.
